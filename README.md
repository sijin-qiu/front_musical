# 音乐剧影视化项目

一个展示音乐剧作品影视化状态的现代化Web应用，基于React框架开发，采用响应式设计。

## 📋 项目概述

本项目是一个信息展示平台，专注于音乐剧作品的影视化进程。用户可以浏览：
- 🏠 **首页** - 项目介绍和统计数据
- 🎬 **已经影视化** - 成功改编为影视作品的音乐剧
- 🎭 **未影视化** - 尚未进行影视化改编的音乐剧作品
- 📝 **筹备影视化** - 正在进行影视化筹备的项目

## 🚀 技术栈

- **前端框架**: React 18
- **路由管理**: React Router DOM 6
- **样式技术**: CSS3 + Flexbox + Grid
- **构建工具**: Create React App
- **字体**: Microsoft YaHei, Arial

## 🛠️ 安装和运行

### 环境要求
- Node.js 16.0+
- npm 或 yarn

### 安装步骤

1. **克隆项目**
```bash
cd 音乐剧
```

2. **安装依赖**
```bash
npm install
```

3. **启动开发服务器**
```bash
npm start
```

4. **浏览器访问**
```
http://localhost:3000
```

### 构建生产版本
```bash
npm run build
```

### 组件说明
- **Navigation**: 顶部导航栏组件，支持响应式菜单

- **NotAdapted**: 未影视化作品展示组件
